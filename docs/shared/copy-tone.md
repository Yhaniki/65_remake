# 文案語氣（Copy & Tone）

> 遊戲內所有文字的风格指南。

## 語氣

- **輕鬆活潑** — 符合跳舞遊戲的歡樂感
- **簡短直接** — 不寫長段落，按鈕 2～4 字
- **繁體中文** — 全站統一
- **不用敬語** — 「你」而非「您」

## 錯誤訊息

| 情況 | 文案 |
|------|------|
| 帳密錯誤 | 帳號或密碼錯誤，請再試一次 |
| 連線失敗 | 無法連線，請檢查網路後再試 |
| 房間已滿 | 這個房間已滿了 |
| 尚未準備 | 還有玩家未準備 |
| 被踢 | 你已被房主移出房間 |
| 伺服器維護 | 伺服器維護中，請稍後再試 |

## 空狀態

| 場景 | 文案 |
|------|------|
| 無房間 | 目前沒有房間，來建立一間吧！ |
| 無歌曲 | 請選擇歌曲 |

## 結算文案（原版用語）

| 元素 | 原版 | 重製建議 |
|------|------|----------|
| 頂部 | 恭喜 您获得 | 可改「本局結果」，保留喜慶感 |
| 表格欄 | 排名、昵称、最高连击、命中率、总积分、成绩 | 繁體：暱稱、最高連擊、命中率、總積分、成績 |
| 奖励 | 总计 120G / 总计 1726 | 繁體：總計 |
| 按鈕 | 确定 / 保存录像 | 繁體：確定 / 保存錄像 |

## 判定文字

| 判定 | 顯示 | 備註 |
|------|------|------|
| Perfect | Perfect | 原版英文 ✅ |
| Cool | Cool | 原版英文 ✅（非 Great） |
| Bad | Bad | 原版英文 ✅ |
| Miss | Miss | 原版英文 ✅ |

> 原版結算用英文判定名。重製可維持英文或改繁中（完美/酷/差/Miss），待決。

## 禁止

- 不用英文 error code 直接展示给玩家
- 不用「哦」「呢」等過度口語
- 不用全形標點混半形
