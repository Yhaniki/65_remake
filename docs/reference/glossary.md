# 名詞表

## 操作相關

| 名詞 | 說明 | 備註 |
|------|------|------|
| 向上 / 向下 / 傾斜 | 音符飛行方向變體 | 每種模式通常有三種方向 |
| 反鍵模式 | 方向鍵映射反轉的模式 | post-MVP |
| 六鍵模式 | 比標準四鍵多兩個鍵的模式 | post-MVP |
| 跳舞毯 | USB 外接腳踏輸入裝置 | MVP 不做 |

## 判定相關

| 名詞 | 說明 | 備註 |
|------|------|------|
| Perfect | 完美判定，最高分 | 原版結算欄位 ✅ |
| Cool | 次完美判定 | 原版結算欄位 ✅（非 Great） |
| Bad | 较差判定 | 原版結算欄位 ✅ |
| Miss | 未按或按錯 | 原版結算欄位 ✅ |
| 最高连击 | 本局 Max Combo | 結算表欄位 |
| 命中率 | 命中百分比 | 結算表欄位，如 99.90% |
| 总积分 | 本局總分 | 結算排名依據 |
| 成绩 | 字母等級 S/A+/A/B… | 結算表欄位 |
| Combo | 連續命中計數 | |
| ShowTime | 特殊高分段落 | 可能是模式也可能是局內事件 |

## 社交相關

| 名詞 | 說明 | 備註 |
|------|------|------|
| 大廳 | 公開區域，顯示頻道與房間列表 | |
| 房間 | 一局遊戲的等待與準備空間 | |
| 房主 | 建立房間的玩家，有踢人/選曲/開始權 | |
| 準備 | 玩家確認 ready 的狀態 | |
| 家族 / 公會 | 玩家組織，有等級與專屬功能 | post-MVP |
| 個人房 | 玩家私人空間，可裝飾、邀請好友 | post-MVP |
| 情侶 | 配對關係，情侶模式需兩人 | post-MVP |

## 經濟相關

| 名詞 | 說明 | 備註 |
|------|------|------|
| 商場 | 購買服裝、道具的商店 | post-MVP |
| G 幣 | 結算獎勵貨幣，顯示如「120G」 | 結算畫面可見 |
| 经验值 | 結算獎勵，升級用 | 結算畫面可見 |
| 点券 / 商城幣 | 商場用貨幣 | post-MVP |

## 技術相關（規劃用）

| 名詞 | 說明 |
|------|------|
| Screen | 玩家看到的整頁介面（登入、大廳等） |
| Flow | 跨多個 screen 的使用者路徑 |
| System | 跨 screen 共用的能力（聊天、判定等） |
| MVP | 最小可玩版本 |
