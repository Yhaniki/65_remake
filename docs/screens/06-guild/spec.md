# 家族公會

> 優先級：**P2 post-MVP**
> 相關 flow：從大廳進入（post-MVP）
> 相關 system：待建立

## 1. 目的

玩家組成家族/公會，有共同等級、專屬活動與社交歸屬感。

- **進入條件**：從大廳點「家族」入口（post-MVP）
- **MVP 狀態**：**不做**，本文檔只留骨架

## 2. 原版行為（Reference）

- 玩家可建立或加入家族/公會
- 家族有等級、成員列表、族長
- 可能有家族專屬頻道或活動
- 可能有家族徽章、專屬服裝

> 待考據。原版細節需截圖或玩家回憶補充。

## 3. 保留（Keep）

post-MVP 再定。MVP 不涉及。

## 4. 改版 / 優化（Change）

| 項目 | MVP | 之後 |
|------|-----|------|
| 整個家族系統 | **不做** | post-MVP 規劃 |
| 大廳入口 | 隱藏 | 開放 |

### MVP 取捨
