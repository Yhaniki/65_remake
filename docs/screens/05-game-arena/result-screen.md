## 1. 目的

- 展示本局所有玩家排名與詳細成績
- 顯示玩家本局獲得的金幣（G）與經驗值
- 提供「確定」回房、「保存錄像」等操作

- **進入條件**：遊戲場地歌曲播放完畢
- **成功離開**：按「確定」→ 回 `04-room`

### UI 元素

| 區塊 | 元素 | 說明 |
|------|------|------|
| 頂部 | 「恭喜 您获得」 | 恭喜文案 |
| 頂部 | 金色 $ 硬幣圖示 | 呼應下方金幣獎勵 |
| 成績表 | 排名欄 | 顯示玩家 **Q 版頭像**（非數字） |
| 成績表 | 昵称 | 玩家名稱 |
| 成績表 | 最高连击 | 本局 Max Combo |
| 成績表 | Perfect / Cool / Bad / Miss | 四級判定 **次數**（非分數） |
| 成績表 | 命中率 | 百分比，如 99.90% |
| 成績表 | 总积分 | 本局總分 |
| 成績表 | 成绩 | 字母等級：**S、A+、A、B…**（橘紅大字） |
| 底部左 | G 幣 | `120G + [ 0 G ]`，銀色 G 幣圖示，下方「总计 120G」 |
| 底部左 | 经验值 | `1726 + [ 0 ]`，下方「总计 1726」 |
| 底部右 | 保存录像 | 圓角粉紅按鈕 |
| 底部右 | 确定 | 圓角粉紅按鈕，回房間 |

### 成績表規則（從截圖推測）

- 最多 **6 人** 一排（截圖 4 人有成績 + 2 空位）
- 依 **总积分** 排序（分高者在上）
- 空位顯示空白列
- 背景：灰階模糊化的 3D 遊戲場地，可見角色剪影

### 判定用詞（原版）

原版結算用 **Perfect / Cool / Bad / Miss** 四欄，**不是** Great / Good。

| 判定 | 結算欄位 |
|------|----------|
| Perfect | ✅ |
| Cool | ✅ |
| Bad | ✅ |
| Miss | ✅ |

## 3. 保留（Keep）

MVP 應保留的原版核心：

- 全員成績 **表格** 形式（多人同屏排名）
- 欄位：排名（頭像）、昵称、最高连击、Perfect/Cool/Bad/Miss、命中率、总积分、成绩
- 底部個人獎勵：**G 幣 + 经验值**（MVP 可簡化數值來源）
- 「确定」回房
- 字母等級（S / A+ / A / B…）
- 模糊化遊戲場地作背景

## 4. 改版 / 優化（Change）

### MVP 必做 vs 延後

| 項目 | MVP | 之後 | 備註 |
|------|-----|------|------|
| 成績表（全欄位） | ✅ | — | 核心 |
| 确定回房 | ✅ | — | 手動按，非自動 |
| 字母成绩 | ✅ | — | |
| G 幣 / 经验值 | 顯示固定值或簡化公式 | 完整經濟 | MVP 不做商城 |
| 保存录像（原版影片） | **不做** | — | 非 osu 式 replay |
| 觀看回放（自由模式） | **不做** | P1 | 本地 `.rpl`；見 [replay-local.md](../../systems/replay-local.md) |
| 恭喜文案 + 硬幣動畫 | 簡化文案即可 | 完整動畫 | |
| 6 人房 | ✅ 支援 | — | 與房間上限一致 |

### 可從他遊戲參考借鑑（非照搬）

| 參考 | 可借 | 不借 |
|------|------|------|
| **Arcaea** | 大分數視覺層級、PURE/FAR/LOST 對比色、右側獎勵區塊 | 單人向布局、角色立繪 |
| **IIDX** | 準確度百分比強調、判定細分列表、performance 折線圖 | 過多技術欄位（FPS 等） |
| **osu!** | 大等級字母、performance 時間軸圖 | 單人向、hex 背景風格 |

### MVP 建議

- **布局**：維持原版 **多人表格**（熱舞核心體驗）
- **視覺**：表格可略現代化（圓角、半透明），但不改成純單人結算
- **操作**：MVP 只保留「确定」；自由模式 P1 加「觀看回放」（非原版「保存录像」）
- **奖励**：MVP 顯示 `+XXX G`、`+XXX 经验` 即可，加成 `[ 0 ]` 可省略

## 6. 資料與狀態

### 進入前（每玩家）

```
PlayerResult {
  userId, displayName, avatarId
  maxCombo: number
  judgments: { perfect, cool, bad, miss }
  accuracy: number      // 0~100，如 99.90
  totalScore: number
  grade: string         // "S" | "A+" | "A" | "B" | ...
}
```

### 離開時（本地玩家額外）

```
Reward {
  goldG: number
  exp: number
}
```

### 排名

- 依 `totalScore` 降序
- `rank` = 1-based 索引

---

## 7. 邊界與錯誤

| 情況 | 處理 |
|------|------|
| 玩家中途斷線 | 仍出現在表內，Miss 偏高、积分偏低 |
| 不足 6 人 | 空列留空或隱藏 |
| 同分 | 待確認：并列 or 比 Perfect 数 |
| 未按确定 | 停留結算頁（不自動跳） |
