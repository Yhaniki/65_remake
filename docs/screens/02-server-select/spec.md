# 伺服器選擇

> 優先級：**P0 MVP**
> 相關 flow：[docs/flows/02-login-to-game.md](../../flows/02-login-to-game.md)
> 相關 system：[docs/systems/account-auth.md](../../systems/account-auth.md)

## 1. 目的

玩家選擇要進入的遊戲伺服器，確認後進入大廳。

- **進入條件**：登入成功
- **成功離開**：選服確認 → 跳轉 `03-lobby`

## 2. 原版行為（Reference）

- 顯示可用伺服器列表
- 每個伺服器可能顯示：名稱、線上人數、狀態（正常/繁忙/維護）
- 點選伺服器 → 確認 → 進入大廳
- 可能有「上次登入」標記
- 維護中的伺服器不可進入

> 待考據：原版「伺服器」和「頻道」是否同一層。見 [open-questions.md](../../reference/open-questions.md)

## 3. 保留（Keep）

- 伺服器列表選擇
- 維護中不可進入
- 選服後進大廳

## 4. 改版 / 優化（Change）

| 項目 | MVP | 之後 | 原因 |
|------|-----|------|------|
| 伺服器數量 | 1 個測試服即可 | 多服 | MVP 驗證流程 |
| 顯示 ping/延遲 | 可選 | P1 | 提升選服體驗 |
| 記住上次伺服器 | 可選 | P1 | 減少重複操作 |
| 自動選推薦服 | 不做 | P2 | MVP 手動選即可 |
| 角色列表預覽 | 不做 | post-MVP | 簡化 |

## 6. 資料與狀態

| 時機 | 資料 |
|------|------|
| 進入前 | userId, sessionToken |
| 離開時 | + serverId, serverName |

## 7. 邊界與錯誤

| 情況 | 處理 |
|------|------|
| 所有伺服器維護中 | 顯示提示，無法進入 |
| 選服後連線失敗 | 顯示錯誤，留在此頁可重試 |
| 伺服器滿員 | 待確認原版是否擋 |
