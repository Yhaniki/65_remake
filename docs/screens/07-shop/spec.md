# 商場

> 優先級：**P2 post-MVP**
> 相關 flow：從大廳進入（post-MVP）
> 相關 system：[docs/systems/avatar-outfit.md](../../systems/avatar-outfit.md)

## 1. 目的

玩家用遊戲內貨幣購買服裝、配件、道具，打扮角色。

- **進入條件**：從大廳點「商場」入口（post-MVP）
- **MVP 狀態**：**不做**，本文檔只留骨架

## 2. 原版行為（Reference）

- 分類瀏覽：髮型、上衣、下裝、鞋子、配件等
- 用金幣或點券購買
- 試穿預覽
- 購買後放入衣櫃，可在換裝介面穿戴

> 待考據。

## 3. 保留（Keep）

post-MVP 再定。MVP 不涉及。

## 4. 改版 / 優化（Change）

| 項目 | MVP | 之後 |
|------|-----|------|
| 整個商場 | **不做** | post-MVP |
| 角色外觀 | 固定預設 1 套 | 開放換裝 |
| 大廳入口 | 隱藏 | 開放 |

### MVP 取捨

MVP 所有玩家使用相同預設外觀（或僅選性別），不涉及貨幣與商城。

### 之後可能方向

- 簡化商城（只有外觀，無課金）
- 或完整還原原版商城 + 點券系統
