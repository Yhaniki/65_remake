# 聊天（Chat）

> 優先級：**P2 post-MVP**（MVP 可省略）
> 使用此 system 的 screen：[03-lobby](../screens/03-lobby/spec.md), [04-room](../screens/04-room/spec.md)

## 職責

- 大廳公開聊天
- 房間內聊天
- 私訊（post-MVP）

## 原版行為（Reference）

- 大廳底部有聊天視窗
- 房間內可聊天
- 可能有頻道聊天、私訊、家族聊天

## MVP 規格

**MVP 不做聊天。** 玩家通过房間列表和準備狀態溝通即可。

## post-MVP 規格（草案）

- 大廳：公開頻道聊天
- 房間：房內聊天
- 訊息格式：`[玩家名]: 訊息內容`
- 基本過濾：禁言詞（待確認）
