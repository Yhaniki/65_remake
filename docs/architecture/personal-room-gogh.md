# 個人房 — 咕盒風

> post-MVP。參考：[咕盒 gogh](https://gogh.gg/zh-sc)

## 定位

| 原版 SDO | 重製（咕盒風） |
|----------|----------------|
| 3D 家具、商城 | **小場景 + 虛擬形象** |
| MMO 裝修 | **氛圍**（BGM、光線） |
| 社交展示 | 陪伴空間 + 可邀友 |

## 功能（post-MVP）

- **虛擬形象**：VMD/MMD idle、簡單動作
- **小房間**：1 背景 + 少量擺件
- **氛圍**：環境音、個人 BGM（含 `.sdm`→ogg）
- **展示**：戰績、skin、最近遊玩
- **社交**：好友參觀、emote（P2）

## 從 gogh 借 / 不借

| 借 | 不借 |
|----|------|
| 小場景一角 avatar | 番茄鐘、生產力 |
| 柔和低干擾 UI | 複雜桌面寵 AI |
| 換裝預覽 | gogh 商城結構 |
